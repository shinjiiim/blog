OFF-APP

Designing a User-Centric Online Shopping App from Scratch Using Design Thinking Methodology

Role UX/UI Designer
Timeline 6 months
Tools Figma, Design Thinking, User Research

Project Overview

OFF-APP is a comprehensive online shopping application designed from the ground up using Design Thinking methodology. The project involved creating a complete user experience with 150+ pages, featuring both Light & Dark Mode capabilities.

The Challenge

Creating a user-centric shopping experience that addresses modern e-commerce needs while maintaining simplicity and usability across all device types.

Design Process

This project followed the Design Thinking methodology, focusing on user needs, iterative design, and comprehensive testing.
