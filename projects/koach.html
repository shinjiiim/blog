Koach

Career Mentorship Platform for Cross-Cultural and Bilingual Users

Role UX/UI Designer, Design Researcher
Company Early-Stage Startup
Timeline 2023 - Present
Tools Figma, Miro, User Research

Project Overview

Koach is an early-stage startup focused on connecting career mentors with mentees, particularly serving cross-cultural and bilingual users who face unique challenges in career navigation and professional development.

My Role & Responsibilities

  • Led UX research to understand the needs of cross-cultural users
  • Designed user flows and wireframes for mentor-mentee matching
  • Created high-fidelity prototypes and design systems
  • Conducted usability testing and iterative design improvements
  • Collaborated with founders on product strategy and feature prioritization

Key Challenges

  • Understanding unique pain points of bilingual and cross-cultural job seekers
  • Designing trust-building features for mentor-mentee relationships
  • Creating an intuitive matching algorithm interface
  • Balancing features for both mentors and mentees in one platform

Solutions & Impact

  • Developed culturally-sensitive user personas and journey maps
  • Designed bilingual interface with seamless language switching
  • Created transparent mentor profile system with verification badges
  • Implemented structured communication tools for effective mentorship

Skills Applied

UX Research: User interviews, Cultural research, Persona development

Design: User flows, Wireframing, Prototyping, Design systems

Collaboration: Stakeholder management, Agile methodology, Cross-functional teamwork
