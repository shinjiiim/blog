UX Research and User Testing for Mobile Applications

Role UX Design Intern
Focus Mobile Applications
Timeline 2021 - 2022
Tools Figma, Maze, User Testing

Project Overview

Focused on UX research and user testing for mobile applications to improve user engagement and satisfaction. Worked on various projects to enhance the mobile user experience through data-driven design decisions.

My Role & Responsibilities

  • Conducted user research interviews and surveys
  • Performed usability testing sessions with mobile app users
  • Analyzed user behavior data and created insights reports
  • Created wireframes and prototypes for mobile interfaces
  • Collaborated with developers to implement design improvements

Key Challenges

  • Identifying and addressing user pain points in existing mobile apps
  • Designing for various mobile device sizes and operating systems
  • Balancing user needs with technical constraints
  • Measuring and improving engagement metrics

Solutions & Impact

  • Improved user satisfaction scores through iterative testing
  • Enhanced mobile app engagement through research-driven redesigns
  • Created standardized testing protocols for consistent evaluation
  • Delivered actionable insights that informed product roadmap

Skills Applied

Research Methods: User interviews, Usability testing, A/B testing, Analytics

Design: Mobile UI, Wireframing, Prototyping, Responsive design

Analysis: Data interpretation, User journey mapping, Heuristic evaluation
